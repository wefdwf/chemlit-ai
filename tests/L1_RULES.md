# ChemLit AI L1 离线测评规则手册

> 每条规则标注对应的 PRD 验收标准，说明检测手段和级别判定依据。
>
> 最后更新：2026-06-21

---

## 通用规则（所有 5 个模块）

### 输出非空

| 属性 | 内容 |
|------|------|
| **级别** | ❌ error |
| **检测方法** | `len(text.strip()) > 10` |
| **对应 PRD** | F2–F6 各模块基础要求：API 返回非空结果 |
| **判级理由** | 纯长度检查，零误报风险 |

### 禁止寒暄确认语

| 属性 | 内容 |
|------|------|
| **级别** | ⚠️ warn |
| **检测方法** | 行首不匹配 `好的|根据您的要求|我将|让我|收到|明白` |
| **对应 PRD** | F3 验收标准第 6 条 / F4 交互规则第 5 条 / F5 验收标准第 5 条：结果中不出现 AI 确认/寒暄语 |
| **判级理由** | 仅检测行首，AI 可在句中写类似表述，存在漏报风险 |

## 主分析（MAIN）

### 1. 包含内容有效性判断

| 属性 | 内容 |
|------|------|
| **级别** | ❌ error |
| **检测方法** | 含 `分析结果` / `## 分析结果` / `无法识别有效文献内容` |
| **对应 PRD** | F2 验收标准第 1 条：返回包含问题回答的结果 |
| **判级理由** | 三种关键词精确匹配，无误报 |

### 2–5. 引导语禁止追加内容（4 条）

| # | 规则 | 级别 | 检测方法 |
|---|------|------|----------|
| 2 | 引导语句号后禁止追加内容 | ❌ error | `建议点击【xxx】` 后无 `，` + `（` 模式 |
| 3 | 禁止引导语括号列举术语 | ❌ error | 引导语后无 `（如xxx）` |
| 4 | 禁止引导语"例如""比如" | ❌ error | 引导语后无 `例如` / `比如` |
| 5 | 引导语后禁止列举术语名 | ❌ error | 引导语后无 `（xxx、xxx）` |

| 属性 | 内容 |
|------|------|
| **对应 PRD** | F2 验收标准第 2、3 条：引导语不出现具体术语名/参数名/缩写，句号后无"例如/比如"或括号列举 |
| **判级理由** | 正则无法穷举所有追加变体，但 PRD 将此列为硬性验收标准 → error |

### 6. 禁止输出子模块完整内容标记

| 属性 | 内容 |
|------|------|
| **级别** | ❌ error |
| **检测方法** | 不同时出现 `### 一句话总结` + `### 创新点` + `### 方法概述` |
| **对应 PRD** | F2 交互规则第 5 条（v6 prompt）：主分析不输出子板块完整内容 |
| **判级理由** | 三标记同时出现才判违规，阈值保守，无误报 |

## 文献速览（OVERVIEW）

### 1. 输出五字段结构

| 属性 | 内容 |
|------|------|
| **级别** | ❌ error |
| **检测方法** | 同时含 `一句话总结` / `创新点` / `方法概述` / `关键结论` / `精读建议` |
| **对应 PRD** | F3 验收标准第 1 条：输出包含上述五个字段 |
| **判级理由** | 五个固定关键词全量匹配，零误报 |

### 2. 精读建议三等级之一

| 属性 | 内容 |
|------|------|
| **级别** | ❌ error |
| **检测方法** | 含 `推荐精读` / `可浏览` / `不建议精读` |
| **对应 PRD** | F3 验收标准第 2 条：明确给出三种等级之一，不接受模糊输出 |
| **判级理由** | 三个固定短语精确匹配，零误报 |

### 3. 方向矛盾检测格式（触发时）

| 属性 | 内容 |
|------|------|
| **级别** | ❌ error |
| **检测方法** | 输出含 ⚠️ 时，必须同时出现：① `你的提问涉及` ② `当前选择的化学分支是` ③ `建议确认` |
| **对应 PRD** | F3 验收标准第 4 条 + PRD 7.3 边界情况：提问与分支矛盾时显示方向矛盾警告 |
| **判级理由** | 仅在 ⚠️ 触发时检查格式三要素，未触发时自动通过。三要素检测为纯关键词匹配 |

### 4–5. 越界检测（2 条）

| # | 规则 | 级别 | 检测方法 |
|---|------|------|----------|
| 4 | 禁止越界输出术语解释 | ⚠️ warn | 无 `术语解释` / `术语名称【方向】` 模式 |
| 5 | 禁止越界输出框架梳理 | ⚠️ warn | 无 `研究问题…研究目标…实验设计/计算方法` 跨行模式 |

| 属性 | 内容 |
|------|------|
| **对应 PRD** | F3 验收标准第 5 条：不包含术语解释、框架梳理或图表解读内容 |
| **判级理由** | 关键词可能出现在正常论述中，存在误报风险 → warn |

## 框架梳理（STRUCTURE）

### 1. 输出包含结论或局限节点

| 属性 | 内容 |
|------|------|
| **级别** | ❌ error |
| **检测方法** | 含 `结论` / `局限` / `展望` |
| **对应 PRD** | F4 验收标准第 1–3 条：三种模板（实验型/计算型/综述型）均以结论/局限/展望收尾 |
| **判级理由** | 三个关键词精确匹配；PRD 三种模板收尾节点固定，无误报 |

### 2. 输出加粗标签

| 属性 | 内容 |
|------|------|
| **级别** | ❌ error |
| **检测方法** | 含 `**` markdown 加粗标记 |
| **对应 PRD** | F4 输出格式：加粗标签，"每个节点写一句话概括" |
| **判级理由** | `**` 是 markdown 精确字符，零误报 |

### 3. 包含文献类型判断分支

| 属性 | 内容 |
|------|------|
| **级别** | ⚠️ warn |
| **检测方法** | 含 `研究问题` / `研究目标` / `文献背景` 等分支关键词 |
| **对应 PRD** | F4 验收标准第 1–3 条：根据文献类型使用对应模板 |
| **判级理由** | 关键词缺失不代表 AI 未做类型判断，只是正则无法从输出反向推断 → warn |

### 4. 文献未体现标注格式（触发时）

| 属性 | 内容 |
|------|------|
| **级别** | ⚠️ warn |
| **检测方法** | "文献未体现"出现时，必须跟在节点标签后（`**节点**：…文献未体现` 或 `节点：…文献未体现`） |
| **对应 PRD** | F4 验收标准第 4 条：文献未体现的节点标注"文献未体现" |
| **判级理由** | 无法从输出判断 AI 应不应该标"文献未体现"（若文献确实覆盖了所有节点则不应标注），所以只在标注了但格式不规范时告警 → warn |

### 5–6. 越界检测（2 条）

| # | 规则 | 级别 | 检测方法 |
|---|------|------|----------|
| 5 | 禁止越界输出术语解释 | ⚠️ warn | 无 `术语名称【方向】` 格式 |
| 6 | 禁止越界输出文献速览 | ⚠️ warn | 不同时出现"一句话总结 + 创新点 + 精读建议" |

| 属性 | 内容 |
|------|------|
| **对应 PRD** | F4 验收标准第 6 条：不包含术语解释、文献速览或图表解读内容 |
| **判级理由** | 关键词可能出现在框架梳理的正常内容中 → warn |

## 术语解释（TERMS）

### 1. 输出术语列表格式

| 属性 | 内容 |
|------|------|
| **级别** | ❌ error |
| **检测方法** | 匹配 `**术语** 【方向】` 格式 |
| **对应 PRD** | F5 验收标准第 3 条：术语标注了所属方向；F5 输出格式：加粗术语 + 方向标注 |
| **判级理由** | 精确正则 `\*\*[^*]+\*\* 【[^】]+】`，零误报 |

### 2. 术语数量 5–10 个

| 属性 | 内容 |
|------|------|
| **级别** | ⚠️ warn |
| **检测方法** | 统计 `**xxx** 【方向】` 出现次数，5 ≤ n ≤ 10 |
| **对应 PRD** | F5 验收标准第 1 条：输出 5–10 个术语 |
| **判级理由** | 边界值（4 或 11）不一定是质量问题，可能是 AI 合理判断 → warn |

### 3–4. 越界检测（2 条）

| # | 规则 | 级别 | 检测方法 |
|---|------|------|----------|
| 3 | 禁止越界输出文献速览 | ⚠️ warn | 不同时出现"一句话总结 + 创新点 + 精读建议" |
| 4 | 禁止越界输出框架梳理 | ⚠️ warn | 无 `研究问题…研究目标…实验设计/计算方法` 模式 |

| 属性 | 内容 |
|------|------|
| **对应 PRD** | F5 验收标准第 4 条：不包含文献速览、框架梳理或图表解读内容 |
| **判级理由** | 关键词可能误匹配 → warn |

## 图表解读（CHART）

### 1. 四段式格式（学术图表时）

| 属性 | 内容 |
|------|------|
| **级别** | ❌ error |
| **检测方法** | 含 `图表类型` / `横坐标` / `关键特征` / `结论` 四个关键词，或含 `该图片不是学术图表`（拒识） |
| **对应 PRD** | F6 处理流程：学术图表按固定四段格式解读 |
| **判级理由** | 四个固定关键词全量匹配 OR 显式拒识，无误报 |

### 2. 禁止使用 markdown 标题

| 属性 | 内容 |
|------|------|
| **级别** | ❌ error |
| **检测方法** | 无 `#` / `##` / `###` 行首标题 |
| **对应 PRD** | F6 输出格式要求：图表解读不使用 markdown 标题（与其他模块区分） |
| **判级理由** | 纯格式正则 `^#{1,3}\s`，零误报 |

### 3. 非学术图表拒识格式（触发时）

| 属性 | 内容 |
|------|------|
| **级别** | ❌ error |
| **检测方法** | 拒识时同时含 `该图片不是学术图表`（标题）+ `请截取文献内的图表进行上传`（副标题） |
| **对应 PRD** | F6 处理流程 + PRD 7.3 边界情况：非学术图表 → 输出标题 + 副标题提示 |
| **判级理由** | 仅在拒识时触发，三要素检测为精确关键词匹配 |

### 4. 包含图表类型判断

| 属性 | 内容 |
|------|------|
| **级别** | ⚠️ warn |
| **检测方法** | 含 `图表类型` / `该图片不是学术图表` |
| **对应 PRD** | F6 处理流程：AI 先判断是否为学术图表 |
| **判级理由** | 关键词缺失不代表 AI 未做类型判断 → warn |

## 级别判定原则

| 级别 | 含义 | 判定标准 |
|------|------|----------|
| ❌ **error** | 必须关注，基本无误报 | PRD 验收标准（硬性要求）+ 检测逻辑精确（纯格式/结构/固定关键词匹配） |
| ⚠️ **warn** | 供参考，可能漏报或误报 | PRD 验收标准或衡量指标 + 检测逻辑不够精确（语义匹配/关键词组合/边界值） |

无法归到 PRD 具体验收标准、仅为代码实现细节的检测项，不纳入 L1 规则。
